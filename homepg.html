Welcome to TripWise

TripWise is a simple web application that helps groups manage and split travel expenses easily. No more confusion about who paid what. No more manual calculations or awkward money talks — just add expenses and get clear summaries instantly.






Features

  • Create a trip
  • Add group members
  • Add expenses
  • Automatic expense splitting
  • View expense summary





How It Works

  1. Create your trip
  2. Add members
  3. Add expenses
  4. View split summary
