# TripWise – Split Your Expenses Wisely

TripWise is a web-based application that helps users manage and split group travel expenses easily.

## Features
- Create Trip
- Add Members
- Add Expenses
- Expense Summary

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
